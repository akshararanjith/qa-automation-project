# Selenium Login Automation Framework

## Overview
This project automates login-related test scenarios using Selenium WebDriver and Pytest.

## Technologies Used
- Python
- Selenium
- Pytest

## Test Cases
1. Valid Login
2. Invalid Login
3. Logout Functionality
4. Login Page Title Validation

## Run Tests

pytest -v

## Project Structure

tests/
- conftest.py
- test_login.py
- test_invalid_login.py
- test_logout.py
- test_page_title.py
